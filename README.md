# Marketing Funnel & Conversion Analysis 

## Project Overview
This project analyzes a real-world bank marketing dataset using Tableau to understand how users move from contact → lead → customer and identify opportunities to improve conversion rates.

## Tools Used
- Tableau
- Excel/CSV dataset
- Funnel Analysis
- Conversion Metrics

## Key Insights
- Overall conversion rate is approximately 11%.
- Significant drop-off occurs after the initial contact stage.
- Cellular channel outperforms telephone in customer conversion.
- Certain job segments show higher subscription rates.
- Longer call duration strongly increases conversion probability.
